# JSONObject

The `JSONObject` scalar type represents JSON objects as specified by [ECMA-404](http://ecma-international.org/publications/files/ECMA-ST/ECMA-404.pdf).

  Based on [graphql-type-json](https://github.com/taion/graphql-type-json).
