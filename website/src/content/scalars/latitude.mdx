# Latitude

A field whose value is a valid
[decimal degrees latitude number](https://en.wikipedia.org/wiki/Latitude) (53.471).

The **input** value can be either in _decimal_ (53.471) or _sexagesimal_ (53Â° 21' 16") format.

The **output** value is always in _decimal_ format (53.471).

The maximum decimal degrees' **precision** is **8**. See
[Decimal Degrees Precision](https://en.wikipedia.org/wiki/Decimal_degrees#Precision) for more
information.
